# marquee-console

Marvin Ivan Mangubat

This is a marquee console implementation with a nostalgic moving ASCII art of a DVD logo. It allows keyboard inputs while the art is flying around the screen. Based on your display's refresh rate, the tick rate at the bottom part of the code should be adjusted.

To run, just open the 'marquee' executable file if you're on a Unix-based machine.
If you're on Windows, please compile the 'marquee.cpp' file first, and then run the corresponding .exe file afterwards.
